# JMath2 Interactive Learning Module

Pure PHP, HTML, CSS, JavaScript, and SQL implementation of a web-based interactive SHS General Mathematics module (Quarter 1 focus) with gamification and analytics.

## Features
- Student registration/login using email or student ID
- Password reset flow
- Admin dashboard and content management
- Quarter -> Module -> Lesson structure
- Pre-test and post-test quizzes (auto-check + randomized items)
- Student progress tracking and history
- XP, levels, badges, and leaderboard
- In-app reminders/notifications
- Responsive UI for mobile and desktop

## Project Structure
- app/: bootstrap, security, repository, database
- public/: front controller + assets
- install/: database installer
- sql/: full schema + seed data
- config/: runtime config generated after install

## Quick Start
1. Serve the project with document root set to public/.
2. Open /install/ and configure MySQL.
3. Open / and login.

## Default Admin
- Username: admin
- Password: Admin12345!

## Local Dev Example
```bash
php -S 127.0.0.1:8080 -t public
```
Then open:
- http://127.0.0.1:8080/install/
- http://127.0.0.1:8080/

## Notes
- config/config.php is generated by installer.
- For production, restrict or remove install/ after setup.
