# JMath2 (Pure HTML/CSS/JS/PHP/MySQL)

Kid-friendly, cartoon-style math learning app with animated bubble landing page, clickable lessons, and quizzes.

## Stack

- HTML
- CSS
- JavaScript
- PHP 8+
- MySQL 8+

## Features Implemented

- Nursery/cartoon UI theme with animated floating bubbles
- Bubble landing page where each bubble opens a lesson
- Quiz flow after each lesson with instant results and explanations
- MySQL installer UI at `/install/`
- URL hardening:
	- Front controller routing via `.htaccess`
	- Request slug sanitization
	- CSRF token verification on quiz submit
	- Secure response headers

## Project Structure

- `public/` - web root (index, assets, route rewrite)
- `app/` - PHP logic (bootstrap, security, DB, repository)
- `install/` - database installer UI
- `sql/` - database schema
- `config/` - runtime config (`config.php` is generated by installer)

## Local Run (Apache + PHP)

1. Point Apache/Nginx document root to `public/`.
2. Ensure Apache `mod_rewrite` is enabled.
3. Open `/install/` and enter MySQL credentials.
4. After successful install, open `/`.

## Notes

- The installer creates `config/config.php` automatically.
- `config/config.php` is gitignored.
- For production, restrict or remove `install/` after setup.
